**Model2(MVC) 기반 상품관리**
상품은 상품아이디,상품명,제조사,가격으로 구성된다 
상품목록을 제공한다 ( 목록은 상품아이디와 상품명만 제공되면 된다 ) 
상품명을 사용자가 클릭하면 상품상세정보가 제공된다 
상품상세정보는 상품아이디,상품명,제조사,가격을 보여준다 
상품정보를 등록할 수 있다
등록시 상품아이디는 자동으로 발급된다 
등록 후 상품목록를 제공한다

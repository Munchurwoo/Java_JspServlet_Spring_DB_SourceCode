**Model2 상품관리**
상품등록 usecase 
: 관리자는 상품명,제조사,가격을 반드시 입력하고
  상품을 등록한다 
  상품아이디와 등록시간은 시스템에서 생성해서
  등록시 함께 저장되어야 한다 
상품목록 usecase 
 : 테이블 형식으로 
   순번과 상품명, 등록시간(년,월,일) 형식으로
   제공한다 
   상품명에는 링크가 제공된다 
상품상세보기 usecase
:  목록에서 상품명을 클릭하면 
   상품아이디를 이용하여 개별 상품을 조회해서 
   상품의 전체 정보를 제공한다 
   상품아이디
   상품명
   제조사
   가격 
   등록 년 월 일 시 분 초 
   로 제공한다
